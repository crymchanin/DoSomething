1) Укажите путь к БД POSTITEM.IB.
2) Выберите IP/имя хоста сервера SQL из списка или пропишите вручную.
3) Укажите логин БД Firebird.
4) Укажите пароль БД Firebird.
5) Измените кодировку при необходимости. Кодировка по умолчанию WIN1251.
6) Для проверки правильности указанных параметров подключения, необходимо нажать кнопку "Тест подключения".

7) Для просмотра данных о режиме работы и адресе подразделения, необходимо нажать кнопку "Получить".
8) Для сохранения резервной копии данных о режиме работы и адресе подразделения, необходимо нажать кнопку "Сохранить" и указать путь для сохранения.

9) Для изменения данных о режиме работы и адресе подразделения, необходимо вручную либо загрузив файл содержаший эти данные заполнить соответствущие текстовые поля в правой нижней части окна программы. После этого нажать на кнопку "Внести изменения на отделении" и подтвердить свое действие.
