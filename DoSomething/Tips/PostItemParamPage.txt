1) Укажите путь к БД POSTUNIT.IB.
2) Выберите IP/имя хоста сервера SQL из списка или пропишите вручную.
3) Укажите логин БД Firebird.
4) Укажите пароль БД Firebird.
5) Измените кодировку при необходимости. Кодировка по умолчанию WIN1251.
6) Для проверки правильности указанных параметров подключения, необходимо нажать кнопку "Тест подключения".

7) Необходимо загрузить список пользователей нажав кнопку "Получить список".
8) Необходимо загрузить список параметров нажав кнопку "Получить список".
9) Для просмотра значения параметра для выбранного пользователя, необходимо нажать кнопку "Получить значение".

10) Чтобы выбранный параметр был персональным для каждого пользователя, он должен быть соответственно помечен. Изменения выполняются нажатием кнопки "Параметр для всех пользователей".
11) Чтобы выбранный параметр был единым для всех пользователей, он должен быть соответственно помечен. Изменения выполняются нажатием кнопки "Параметр для одного пользователя".

12) Для изменения значения параметра для текущего пользователя/всех существующих пользователей (если установлен флажок "Для всех пользователей"), необходимо нажать кнопку "Изменить значение".
